# 智易通(ZhiYiTong) 数据库设计说明

## 1. 设计概述

### 1.1 设计目标
- 构建多租户架构的财务管理系统
- 支持多会计制度及其切换
- 集成AI智能化功能
- 确保数据安全性和完整性
- 提供灵活的扩展能力

### 1.2 设计原则
1. **数据隔离**
   - 系统级/企业级数据分离
   - 多租户数据严格隔离
   - 共享数据统一管理

2. **标准化**
   - 统一的命名规范
   - 标准的字段定义
   - 规范的表结构设计

3. **安全性**
   - 多级权限控制
   - 操作日志记录
   - 敏感数据加密

4. **扩展性**
   - 支持功能模块扩展
   - 预留字段满足未来需求
   - 版本控制支持平滑升级

### 1.3 命名规范

#### 1.3.1 数据库命名
- 系统主库：`zyt_sys`
- 企业账套库：`zyt_company_编号`（如：zyt_company_001）

#### 1.3.2 表命名前缀
- 系统表：sys_
- 账务表：account_
- AI相关：ai_
- 基础表：base_

#### 1.3.3 字段命名规范
- 主键：id (INT AUTO_INCREMENT)
- 创建信息：created_at, created_by
- 更新信息：updated_at, updated_by
- 状态字段：status (1:正常 0:禁用)
- 外键引用：表名_id
- 代码字段：code
- 名称字段：name
- 备注说明：description

## 2. 数据库架构

### 2.1 系统主库(zyt_sys)
系统主库存储全局配置和共享数据，包括：
- 企业信息管理
- 会计制度管理
- 科目模板管理
- 用户权限管理
- AI模型配置

### 2.2 企业账套库(zyt_company_xxx)
每个企业独立数据库，包括：
- 基础数据
- 凭证管理
- 账簿管理
- 报表数据
- AI应用数据

## 3. 初始化流程

### 3.1 账套创建流程
1. 创建账套记录
   ```sql
   INSERT INTO sys_company (company_code, company_name, ...) VALUES (...);
   ```

2. 创建账套数据库
   ```sql
   CREATE DATABASE zyt_company_xxx DEFAULT CHARACTER SET utf8mb4;
   ```

3. 创建账套表结构
   ```sql
   -- 执行账套表结构初始化脚本
   SOURCE 02_create_company_db.sql;
   ```

4. 初始化基础数据
   ```sql
   -- 从会计制度模板复制科目
   INSERT INTO account_subject SELECT * FROM sys_subject_template WHERE system_id = xxx;
   ```

5. 设置会计期间
   ```sql
   -- 创建会计期间
   INSERT INTO accounting_period (year, period, ...) VALUES (...);
   ```

### 3.2 初始化数据
1. 科目表初始化
   - 复制制度模板科目
   - 设置科目层级关系
   - 配置辅助核算

2. 会计期间设置
   - 设置会计年度
   - 创建会计期间
   - 设置当前期间

3. 基础数据配置
   - 常用摘要
   - 辅助核算项目
   - 系统参数

## 4. 表结构说明

### 4.1 系统主库表
1. 企业信息表(sys_company)
   - 企业基本信息
   - 账套配置信息
   - 会计制度设置

2. 会计制度表(sys_accounting_system)
   - 制度基本信息
   - 版本管理
   - 生效日期

3. 科目模板表(sys_subject_template)
   - 科目基本信息
   - 科目层级关系
   - 辅助核算设置

4. 用户权限相关表
   - 用户表(sys_user)
   - 角色表(sys_role)
   - 权限表(sys_permission)

5. AI配置表
   - 模型配置(sys_ai_config)
   - 知识库(sys_ai_knowledge)
   - 使用记录(sys_ai_usage_log)

### 4.2 账套数据库表
1. 科目表(account_subject)
   - 科目信息
   - 层级关系
   - 辅助核算

2. 凭证相关表
   - 凭证主表(voucher)
   - 凭证分录表(voucher_entry)
   - 凭证附件表(voucher_attachment)

3. 期间相关表
   - 会计期间表(accounting_period)
   - 期间结转记录(period_closing)

4. 辅助核算表
   - 辅助核算类型(auxiliary_type)
   - 辅助核算项目(auxiliary_item)

## 5. 数据维护

### 5.1 数据备份
```sql
-- 备份系统主库
mysqldump -u admin -p zyt_sys > zyt_sys_backup.sql

-- 备份企业账套库
mysqldump -u admin -p zyt_company_xxx > company_xxx_backup.sql
```

### 5.2 数据恢复
```sql

mysql -u admin -p zyt_sys < zyt_sys_backup.sql
```
