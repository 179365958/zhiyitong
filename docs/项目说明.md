# 智易通(ZhiYiTong)小微企业财务软件

> 智能财务，易如反掌

## 品牌说明
- 中文名称：智易通
- 英文名称：ZhiYiTong
- 英文简称：ZYT
- 产品定位：AI驱动的智能财务管理系统
- 核心价值：通过AI技术赋能，让小微企业的财务管理变得简单、智能、高效

## 1. 项目概述

### 1.1 项目目标
开发一款适用于中国小微企业的财务软件，特点是易用性高、部署简单、符合中国会计准则。

### 1.2 技术栈
- 前端：Vue 3 + Element Plus + ECharts
- 后端：Node.js + Express
- 数据库：MySQL/SQL Server（双数据库支持）

## 2. 功能模块

### 2.1 系统管理
1. 账套管理
   - 账套创建向导
     * 基本信息设置
     * 会计制度选择
     * 会计期间设置
     * 本位币设置
   - 账套列表管理
   - 账套初始化进度
   - 账套切换功能
   - 数据库初始化

2. 会计制度管理
   - 制度维护
   - 科目模板管理
   - 制度切换功能
   - 版本管理

3. 权限管理
   - 账套级权限
   - 功能级权限
   - 数据级权限
   - 操作级权限

### 2.2 核心功能
1. 凭证管理
   - 凭证录入（手工/导入）
   - 凭证审核
   - 凭证查询/打印

2. 账簿管理
   - 总账
   - 明细账
   - 科目余额表

3. 辅助核算
   - 客户/供应商管理
   - 部门/职员管理
   - 项目/存货管理
   - 多维度统计分析
   - 辅助核算余额表
   - 辅助核算明细账

4. 报表管理
   - 资产负债表
   - 利润表
   - 现金流量表

5. 期末处理
   - 结账
   - 结转损益

### 2.3 扩展功能
1. 往来管理
   - 应收账款
   - 应付账款

2. 固定资产
   - 资产卡片
   - 折旧计提

3. 票据管理
   - 发票管理
   - 开票/收票记录

## 3. 数据库设计

### 3.1 数据库架构
1. 系统主库(zyt_sys)
   - 企业信息管理
   - 会计制度管理
   - 科目模板管理
   - 用户权限管理
   - AI模型配置

2. 企业账套库(zyt_company_xxx)
   - 基础数据
   - 凭证管理
   - 账簿管理
   - 报表数据
   - AI应用数据

### 3.2 初始化流程
1. 账套创建
   - 创建账套数据库
   - 创建账套表结构
   - 初始化基础数据
   - 设置会计期间

2. 基础数据初始化
   - 科目表初始化
   - 会计期间设置
   - 辅助核算项目
   - 常用摘要
   - 基础参数配置

### 3.3 数据库特点
- 支持MySQL 5.6+
- 支持SQL Server
- 完整的事务支持
- 数据一致性保证

## 4. API设计

### 4.1 账套管理
```javascript
// 账套管理
POST /api/company/create          // 创建账套
GET  /api/company/list           // 获取账套列表
GET  /api/company/init-progress  // 获取初始化进度
POST /api/company/switch         // 切换账套

// 会计制度
GET  /api/accounting-system/list    // 获取会计制度列表
GET  /api/accounting-system/detail  // 获取制度详情
POST /api/accounting-system/switch  // 切换会计制度
```
